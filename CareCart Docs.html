Frontend Setup (Flutter/Web) 


Welcome to CareCart, your multi-platform e-commerce solution powered by Flutter. This guide will walk you through the steps required to set up and configure the CareCart app on your Linux, MacOS or Windows system.


### System Requirements


Before you begin, ensure your system meets the following requirements:


- MacOS or Windows or Linux operating system

- Flutter SDK

- Android Studio/Xcode (Optional But recommended) or VSCode

- Web Browser

### Installation


Follow these steps to install and configure CareCart:


1. **Install Flutter**:

    - Download Flutter from the [official website](https://flutter.dev/docs/get-started/install).

    - Follow the installation instructions based on your operating system.



2. **Run Backend Project**:

Ensure your FastAPI backend is running on localhost port 8000 before starting your Flutter project.



3. **Generate Code for JsonSerializable**:

Generate code for JSON serialization using the `build_runner` package. This command will build the necessary files for your models to handle JSON serialization and deserialization.

flutter pub run build_runner build



3. *Run Flutter Project**:

Run your Flutter project on your desired platform (web, Android, iOS). For development purposes, you can start by running it on a web emulator.

flutter run -d chrome

 



Backend Setup with FastAPI


#### 1. Setup Environment

First, create a virtual environment to isolate your project's dependencies and avoid conflicts with other projects.


```bash

python3 -m venv .venv

source .venv/bin/activate

```


#### 2. Install Libraries

Next, install the required libraries listed in your `requirements.txt` file. This file should include all the dependencies your FastAPI project needs.


```bash

pip3 install -r ./requirements.txt

``` 


#### 3. Seed Json Data to DB

CareCart/backend/populate_db.py



#### 4. Start the Server

Finally, start the FastAPI server using Uvicorn. This command will run your server on `http://0.0.0.0:8080` and enable auto-reload for development purposes.


```bash

python3 -m uvicorn main:app --reload --host 0.0.0.0 --port 8080

```
